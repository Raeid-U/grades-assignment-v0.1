# Grades Calculator

## Description

- Series of prompts in Console/Terminal based in C++
- Conducts Calculations for GPA, Average, Final Grade, and Required Marks

### Dependencies

- Standard C++ Libraries
- Windows 10, Mac OS "Sonoma", Linux Mint 21.3 "Virginia"
- bits/stdc++.h
